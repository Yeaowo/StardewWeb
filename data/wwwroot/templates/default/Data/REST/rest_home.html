Today

Pulls the same data as on the Today page.
  • /api/v1/Today/Tasks: Get a list of today's Tasks
  • /api/v1/Today/Mail: Get a list of unread mail
  • /api/v1/Today/Events: Get a list of today's Events
  • /api/v1/Today/News: Get a list of today's News
  • /api/v1/Almanac/Crops: Get a list of in season crops
  • /api/v1/Almanac/Fish: Get a list of in season fish

Farm Assets

  • /api/v1/Assets/Crops: Get a list of all crops growing
  • /api/v1/Assets/LiveStock: Get a list of all live stock
  • /api/v1/Assets/Machines: Get a list of all machines
  • /api/v1/Assets/Buildings: Get a list of all buildings
  • /api/v1/Assets/Inventory: Get a list of all items in inventory

Catalogue

Pulls lists of in game items.
  • /api/v1/Catalog/All: Get a list of game Items
  • /api/v1/Catalog/Items: Get a list of game Objects
  • /api/v1/Catalog/Items/Raw: Get a list of game Objects without string replacement
  • /api/v1/Catalog/Tools: Get a list of Tools
  • /api/v1/Catalog/Tools/Raw: Get a list of Tools without string replacement
  • /api/v1/Catalog/Shirts: Get a list of Shirts
  • /api/v1/Catalog/Shirts/Raw: Get a list of Shirts without string replacement
  • /api/v1/Catalog/Pants: Get a list of Pants
  • /api/v1/Catalog/Pants/Raw: Get a list of Pants without string replacement
  • /api/v1/Catalog/Weapons: Get a list of Weapons
  • /api/v1/Catalog/Weapons/Raw: Get a list of Weapons without string replacement

Growables

Pulls lists of items that can be grown.
  • /api/v1/Growables/Crops: Get a list of crops with extended information
  • /api/v1/Growables/Crops/Raw: Get a list of the raw CropData entries
  • /api/v1/Growables/GiantCrops: Get a list of Giant Crop entries
  • /api/v1/Growables/FruitTrees: Get a list of Fruit Tree entries
  • /api/v1/Growables/FruitTrees/Raw: Get a list of Fruit Tree entries without string replacement

Crafting

Pull lists of items that can be crafted.
  • /api/v1/Crafting/Crafttables: Get a list of all crafting recipes
  • /api/v1/Crafting/BigCrafttables: Get a list of all Big Craftable recipes
  • /api/v1/Crafting/BigCrafttables/Raw: Get a list of Big Craftable recipes without string replacement

Locations

Pull location lists and details.
  • /api/v1/Locations/List: Get a list of all game Locations
  • /api/v1/Locations/Summary: Get a summary list of all game Locations
  • /api/v1/Locations/Details/{locationName}: Get the details of the Location

Mail

Pulls lists of Mail messages
  • /api/v1/Mail/All: Get a list of all mail received Ids
  • /api/v1/Mail/All/Details: Get a detailed list of mail received
  • /api/v1/Mail/New: Get a list of new mail Ids
  • /api/v1/Mail/New/Details: Get a detailed list of new mail
  • /api/v1/Mail/Tracking: Get a list of tracking mail Ids
